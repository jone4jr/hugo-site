### Detailed Course Structure

**Section 1: Define Phase (4 hours)**
- Project selection criteria
- Creating effective project charters
- SIPOC diagrams
- Voice of the Customer (VOC)
- Critical to Quality (CTQ) characteristics
- Stakeholder analysis
- Project planning and scheduling

**Section 2: Measure Phase (8 hours)**
- Types of data and measurement scales
- Data collection planning
- Sampling strategies
- Measurement Systems Analysis (MSA)
- Gage R&R studies (variable and attribute)
- Process mapping in detail
- Baseline process capability
- Calculating Cp, Cpk, Pp, Ppk

**Section 3: Analyze Phase (8 hours)**
- Graphical analysis techniques
- Descriptive statistics
- Probability distributions
- Hypothesis testing fundamentals
- t-tests (1-sample, 2-sample, paired)
- ANOVA (1-way and 2-way)
- Chi-square tests
- Correlation analysis
- Simple and multiple regression
- Root cause verification

**Section 4: Improve Phase (6 hours)**
- Generating solutions
- Solution selection matrices
- Design of Experiments (DOE)
- Full factorial designs
- Fractional factorial designs
- Response surface methodology
- Pilot planning and execution
- Implementation planning
- Cost-benefit analysis

**Section 5: Control Phase (4 hours)**
- Control plan development
- Statistical Process Control (SPC)
- Selecting appropriate control charts
- Variable control charts (X-bar R, X-bar S, Individuals)
- Attribute control charts (p, np, c, u)
- Mistake-proofing (Poka-Yoke)
- Documentation and standardization
- Transition to process owner

**Section 6: Lean Integration (4 hours)**
- Value stream mapping
- Identifying and eliminating waste
- Takt time calculations
- Kanban system design
- SMED methodology
- 5S advanced application
- Kaizen event facilitation

**Section 7: Project Management & Leadership (3 hours)**
- Team formation and dynamics
- Meeting facilitation
- Stakeholder communication
- Change management
- Overcoming resistance
- Presenting to leadership

**Section 8: Case Studies & Certification (3 hours)**
- Complete real-world case studies
- Practice problems
- Review and synthesis
- Certification exam preparation
- Final exam (80% passing required)
